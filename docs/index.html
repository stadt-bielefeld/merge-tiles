convertImage

Converts an image in a list other formats.

convertImage(file: string, formats: array, callback: Function)
Parameters
file (string) Path of the input file
formats (array) Array of output formats like [ 'gif','tif','jpg','png' ]
callback (Function) Callback like (err) => {}
Example
let convertImage = require('merge-tiles').convertImage;

convertImage(__dirname + '/output/all.png', ['tif', 'gif', 'jpg'], (err) => {
  if (err) {
    console.log(err);
  }
});

isImage

Checks, if the file is a supported image.

isImage(file: string): (ImageInfo | Boolean)
Parameters
file (string) Filename of a wms-downloader tile.
Returns
(ImageInfo | Boolean): Unsupported image: false, supported image: Object like { ext: 'png', wfExt: 'pgw' }
Example
isImage('x1y2.png'); // => { ext: 'png', wfExt: 'pgw' }
isImage('x1y2.svg'); // => false

mergeTiles

Merges wms-downloader tiles to a single tile.

mergeTiles(inputDir: string, outputDir: string, workers: number, callback: function)
Parameters
inputDir (string) Input directory of wms-downloader tiles.
outputDir (string) Output directory of single tile.
workers (number) Count of graphicsmagick workers.
callback (function) Callbackfunktion like this: function(err){}
Example
let mergeTiles = require('merge-tiles').mergeTiles;

mergeTiles(__dirname + '/input', __dirname + '/output', 1, (err) => {
  if(err){
    console.log(err);
  }
});

compareFilesOnX

That compareFunction can be used to sort objects of filenames in an array. It compares filename objects on their x attribute.

compareFilesOnX(a: object, b: object): number
Parameters
a (object) Filename object a
b (object) Filename object b
Returns
number: 0: equal, 1: a.x is bigger as b.x, -1: a.x is smaller as b.x

That compareFunction can be used to sort objects of filenames in an array. It compares filename objects on their y attribute.

compareFilesOnY(a: object, b: object): number
Parameters
a (object) Filename object a
b (object) Filename object b
Returns
number: 0: equal, 1: a.y is bigger as b.y, -1: a.y is smaller as b.y

FileInfo

A wms-downloader tile has a filename like this: x0y1.png This object contains the value after x and after y as integer value.

FileInfo

Type: Object

Parameters
file (any)
Properties
x (string) : X value of wms-downloader tile.
y (string) : y value of wms-downloader tile.
Example
//wms-downloader tile: x0y1.png
{ x: 0, y: 1 }

determineXY

Determines the x and y value as integer from filename.

determineXY(file: string): FileInfo
Parameters
file (string) Filename of a wms-downloader tile.
Returns
FileInfo: Object with x and y values as integer like {x: 0, y: 1}.
